This is a set of bash tools to manipulate bitcoin addresses.

https://github.com/grondilu/bitcoin-bash-tools

## SYNOPSIS

    $ git clone https://github.com/grondilu/bitcoin-bash-tools.git
    $ . ./bitcoin-bash-tools/bitcoin.sh
    $ newBitcoinKey
